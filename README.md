## Инструкция

1. Скачать HykkitAgent-1.0.0.jar (из releases)
2. Закинуть в корневую папку сервера Hytale
3. В запускаторе (.bat/.sh) прописать: `-javaagent:HykkitAgent-1.0.0.jar`

## Принцип работы

Патчер создан для простой цели: подмены метода первичной проверки пакета игрока при его входе. Полностью игнорируется проверка токена игрока и сервера, а также версии игры.
Написан с помощью **ByteBuddy**
